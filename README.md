## Excellent Mentor Dual Selection System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/5e4ee1f3176246ada2f4c798555fb683/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular users, mentors, administrators.

**Regular User Features:**
1. Login
2. Registration
3. Change Password
4. Modify Information
5. Mentor Information
6. Mentor Selection
7. Project Information Management
8. Project Submission Management
9. Mentorship Project Management



**Mentor Features:**
1. Login
2. Change Password
3. Modify Information
4. Mentor Selection
5. Project Information Management
6. Project Submission Management
7. Mentorship Project Management

**Administrator Features:**
1. Login
2. Change Password
3. Personal Information
4. Student Management
5. Mentor Management
6. Mentor Information Management
7. Mentor Selection Management
8. Project Information Management
9. Project Submission Management
10. Mentorship Project Management
